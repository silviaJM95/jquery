jQuery 的简介 1、优势： 轻量 强大的选择器 出色的DOM操作的封装 可靠的事件处理机制 完善的Ajax 不污染顶级变量 出色的浏览器兼容性 链式操作方式 隐式迭代 行为层与结构层的分离 丰富的插件支持 完善的文档 开源 #########注意点############## 1、为jQuery代码写注释 2、jQuery对象不能使用DOM对象的任何方法，例如：$('#id').innerHTML，$('#id').checked是错误的 可以使用$('div').html() 和 $('div').attr('checked') DOM对象也不能使用jQuery里的方法 3、如果获取的是jQuery对象，在获取变量的时候，要在变量上加$ 4、jQuery对象提供了两种方式将一个jQuery对象转换为DOM对象 1）jQuery对象是一个类数组对象，可以通过 index 方法得到相应的DOM对象 var $cr = $('#cr'); var cr = $cr[0] 2）jQuery本身提供的 get(index) 方法得到相应的DOM对象 var $cr = $('#cr'); var cr = $cr.get(0); 5、DOM 对象转成 jQuery对象 ————> $(DOM对象) $()函数是一个jQuery对象的制造工厂 6、Dreamweaver是建立在Web站点和应用程序的专业工具 7、选择器中含有特殊字符，“· 、 # （ ]”在使用jQuery获取的时候使用\\转义字符
